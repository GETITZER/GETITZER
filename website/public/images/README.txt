====================================

the moment the files are present (PNG/JPG fallback to SVG if missing).

  logo-white.png     → White version for dark backgrounds (optional, falls back to logo.png)
  favicon.png        → Square icon (512×512 recommended)
  about.jpg          → About section background/hero (1920×1080)
  hero-bg.jpg        → Main hero background (1920×1080, optional)
